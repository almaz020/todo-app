# 📅 Todo Daily Planner

Android-приложение — ежедневник с календарём и задачами по часам.

---

## ✨ Функционал

* 📆 Горизонтальный календарь с выбором даты
* 🕐 Отображение задач по часам (00:00–23:00)
* ➕ Создание задачи с выбором даты и времени
* 📋 Экран деталей задачи
* 💾 Локальное хранение (Room)
* 🔄 Автообновление списка задач

---

## 🧱 Архитектура

Проект построен с использованием:

* **MVVM + Clean Architecture**
* Многомодульная структура:

  * `core:database`
  * `feature:tasks`
* Разделение на слои:

  * `data`
  * `domain`
  * `presentation`

---

## 🛠️ Технологии

* Kotlin
* Jetpack Compose
* Navigation Compose
* Room
* Koin (DI)
* Gson

---

## 🧠 Особенности реализации

* Задачи распределяются по часовым слотам (24 часа)
* Корректная обработка пересечений времени:

  * задача 09:00–10:00 отображается только в одном слоте
* Начальные данные загружаются из JSON в Room
* Реализована валидация времени при создании задачи

---

## 🧪 Тестирование

Добавлены unit-тесты:

* Проверка фильтрации задач по дню
* Проверка корректного распределения задач по временным слотам

---

## 📱 Требования

* Android 8.0+ (API 26)
* Портретная ориентация
